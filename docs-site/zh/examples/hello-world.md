# Hello World

每个程序员的旅程都从同一行字开始。在 1y 中,让计算机说声"你好"只需要一行代码——但这一行背后藏着这门语言最朴素也最核心的承诺:**一切皆表达式,默认不可变,数值不会溢出**。本例从最简单的输出开始,逐步扩展成一个读取用户输入、用插值字符串生成问候的小程序,带你熟悉 1y 程序的基本骨架。

## 最简版本

我们从一行代码开始:

```1y
println("Hello, World!")
```

把这行保存为 `hello.1y`,然后在命令行运行:

```
1y run hello.1y
```

你会看到终端打印出 `Hello, World!`。

这一行里发生了什么?`println` 是一个全局内置函数,不需要任何 `import` 就能使用。它接受任意值,把它转换成可显示的字符串并追加一个换行,再写到标准输出。`"Hello, World!"` 是一个字符串字面量,双引号包裹。这就是 1y 程序最小的可运行单元——一个表达式语句。

## 把问候封装成函数

直接写死问候语不够灵活。真实程序里,我们通常希望问候能根据调用者传入的名字动态变化。在 1y 中定义函数用的是 `fn` 关键字:

```1y
fn greet(name) -> Str {
    "Hello, {name}!"
}

println(greet("World"))
println(greet("1y"))
```

让我们逐行拆解:

- `fn greet(name) -> Str` 声明一个名为 `greet` 的函数,它接受一个参数 `name`,返回类型标注为 `Str`。1y 是动态类型语言,类型标注主要起文档作用,帮助你与未来的读者(包括你自己)沟通意图。
- 函数体只有一行:`"Hello, {name}!"`。注意这里出现的 **字符串插值**——`{name}` 会被替换成变量 `name` 的当前值。无需 `+` 拼接,也无需 `format` 模板,插值直接嵌在字符串字面量里。
- 因为 1y 是表达式导向的语言,函数体最后一个表达式的值就是函数的返回值,不需要显式写 `return`。`"Hello, {name}!"` 既是函数体的最后一行,也是 `greet` 的返回值。

运行后你会看到:

```
Hello, World!
Hello, 1y!
```

::: tip 字面量花括号怎么写?
由于 `{` 和 `}` 在字符串里被用作插值标记,如果你需要打印一个真正的花括号,用反斜杠转义:`"\{ not a variable \}"`。
:::

## 加入用户输入

真正的程序往往要和人交互。我们让用户输入自己的名字,程序再回以问候。读取标准输入需要用到 `io` 模块:

```1y
import io;

fn greet(name) -> Str {
    "Hello, {name}!"
}

print("What is your name? ");
let name = io.read_line();

match name {
    s if is_str(s) => println(greet(trim(s))),
    nil => println("Hello, silent friend!")
}
```

这段程序引入了几个新概念,我们逐一说明。

**`import io;`** 把标准库的 `io` 模块引入当前作用域,绑定到名字 `io`。1y 的模块系统很直接:被引入的模块成为一个命名空间,通过 `io.read_line` 这样的点号访问其内部函数。`import` 默认是立即加载;如果你希望推迟到第一次使用时再加载,可以写成 `lazy import io;`,这对加快启动速度很有用。

**`print`** 与 `println` 的唯一区别是它不追加换行。我们用它打印提示语 `What is your name? `,让光标停在问号之后等待输入,视觉上更自然。

**`io.read_line()`** 从标准输入读取一行文本。当输入结束时(例如用户按下 Ctrl+D / Ctrl+Z),它返回 `nil`。这是一个"可能失败"的操作,1y 用 `nil` 表示缺席的值,而不是抛出异常,因为读到 EOF 在交互式程序里属于正常的结束路径。

**`let name = io.read_line();`** 把读取结果绑定到不可变变量 `name`。注意末尾的分号——它表示这是一个"语句",其值被丢弃;而块的最后一条不带分号的表达式会成为整个块的值。这是 1y 区分"语句"与"值表达式"的细微之处。

**`match name { ... }`** 是 1y 处理"可能缺席的值"的惯用方式。我们用模式匹配对 `name` 分情况处理:

- `s if is_str(s)` 匹配任何字符串,并用守卫 `is_str(s)` 确保它确实是字符串(而非 `nil`)。绑定 `s` 指向被匹配的值。`is_str` 是内置的类型谓词,1y 还提供 `is_int`、`is_map` 等一系列同类函数。
- `nil` 匹配输入结束的情形,给出一个兜底的问候。
- 注意我们对 `s` 调用了 `trim(s)`——`read_line` 返回的字符串末尾通常带着换行符 `\n`,去掉它才能得到干净的名字。`trim` 是内置字符串函数,会剥掉首尾空白。

运行时,程序会停下来等你输入:

```
What is your name? Ada
Hello, Ada!
```

## 小结

这个例子虽然短小,却涵盖了 1y 程序的几块基石:用 `println` 输出、用 `fn` 定义函数、用 `{name}` 做字符串插值、用 `import` 引入模块、用 `let` 绑定值、用 `match` 分情况处理。更重要的是,它体现了 1y 的表达风格——没有样板代码,没有多余的 ceremony,程序读起来就像一份对意图的直接陈述。接下来,我们将用同样的方式逐步构建更复杂的例子,从数值计算到并发编程。
